Wireless MIDI (Bluetooth MIDI) – The Cable-Free Setup

Wireless MIDI lets you connect a MIDI keyboard (or other MIDI gear) to an iPad, Mac, or Windows computer without a traditional MIDI cable. If you’ve been using MIDI cables for decades, this can feel like a genuine upgrade.
