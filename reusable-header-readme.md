# Reusable Header System Applied

This site now uses:
- `shared/site-shell.css`
- `shared/site-chrome.js`

Each HTML page contains:
- `<div id="site-header"></div>`
- `<div id="site-footer"></div>`

So future menu/footer edits only need to happen in:
- `shared/site-chrome.js`
